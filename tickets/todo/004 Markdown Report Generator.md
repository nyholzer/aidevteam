# Feature: Markdown Report Generator
## Description

Create a script generate_report.py in projects/repo_analyzer_v1/src/ that reads the full_repo_index.json and creates a summary for the PM (Human).

## Requirements

    Read the master index.

    Create a file projects/repo_analyzer_v1/SUMMARY.md.

    Include a table with: Path, Extension, and Size.

    Add a "Statistics" section at the top showing the Total Number of Files and Total Project Size.
